# aptFlow

O projeto tem como objetivo simular o movimento do ar em um ambiente tridimensional, permitindo a visualização do fluxo de ar e a interação com o ambiente.
O projeto utiliza cmake e OpenGL para renderização gráfica, além de outras bibliotecas como GLFW e GLM para facilitar o desenvolvimento. O projeto envolve prática de conceitos e práticas da programação, como paralelismo, organização de código, versionamento por meio do git, caching, entre outros.
